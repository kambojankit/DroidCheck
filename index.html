DroidCheck

DroidCheck is an Application dedicated to analysis of Android APKs, to differentiate between benign and malicious Android Application based upon their attributes and behaviour.

The application does static analysis on the APKs to find the permission used by the application. The application also does behaviour analysis of the APKs, to detect behavioural patterns of malicious applications
